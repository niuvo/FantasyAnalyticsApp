# Fantasy Football Analytics Application

## Overview
Fantasy Football Analytics Application is a data-driven tool designed to assist fantasy football enthusiasts in drafting NFL players. The application integrates real-time player data and utilizes OpenAI's advanced analytics for in-depth analysis and insights.

## Features
- Real-time data fetching from the ESPN API.
- Data analysis using the OpenAI API.
- Insights for fantasy football drafting.
- A user-friendly interface for interacting with the analytics.

## Tech Stack
- **Backend**: Python, Flask
- **Database**: MySQL
- **Containerization**: Docker
- **Orchestration**: Kubernetes
- **Frontend**: HTML, CSS, JavaScript
- **APIs**: ESPN API, OpenAI API
- **Version Control**: Git
